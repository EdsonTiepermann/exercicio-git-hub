O PHP foi criado em 1995 pelo programador canadense Rasmus Lerdorf. A sigla é um acrônimo para PHP "Hypertext Preprocessor" A linguagem tornou-se popular por ter uma sintaxe mais simples, e por mesclar o código executado no lado do servidor com HTML, facilitando a criação de páginas com conteúdo dinâmico. um script mais automatizado.

No curso vamos usar o PHP como um script, igual o javascript, , mas com o PHP nos possibilita criar um servidor mais automatizado como validar um formulário antes mesmo de serem enviados ao browser que você utiliza.
